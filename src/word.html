Microsoft Word:

O Microsoft Word é um processador de texto produzido pela Microsoft Office/Microsoft 365. Foi criado por Richard Brodie para computadores IBM PC com o sistema operacional DOS em 1983. Mais tarde foram criadas versões para o Apple Macintosh, SCO UNIX e Microsoft Windows.
